# Nomisma

<img class='project-main' src='{{site.baseurl}}/assets/nomisma-logo.png'/>


Nomisma is a currency convertor application , useful for currency conversion needed in daily life.

The application was built as an alternative to information intensive currency conversion applications, often used for financial tasks.

The application is built using Javascript + React and hosted using Github Pages.

Conversion rates are fetched on page load from the fixer.IO API. 

A live demo of this project is available on this site <a target="_blank" href="http://niamhlawlor.com/nomisma-web/">HERE</a>
